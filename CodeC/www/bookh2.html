PARAMETER

(href) Specifies the destination address. It can be an absolute or relative URL, or the name of an anchor. An absolute URL is the complete URL of a website like http://example.com/. A relative URL points to another directory and/or document inside the same website, e.g. /about-us/ points to the directory “about-us” inside the root directory (/). When pointing to another directory without explicitly specifying the document, web servers typically return the document “index.html” inside that directory.

hreflang?

The hreflang Specifies the language of the resource linked by the href attribute (which must be present with this one). Use language values from BCP 47 for HTML5 and RFC 1766 for HTML 4.

(rel) Specifies the relationship between the current document and the linked document. For HTML5, the values must be defined in the specification or registered in the Microformats wiki.



Who invented HTML

HTML (Hypertext Markup Language) was developed by Tim Berners-Lee, a physicist and contractor at CERN, in the late 1980s and early 1990s. Berners-Lee proposed and prototyped a hypertext system called ENQUIRE for CERN researchers to share and use documents [0]. In 1989, he wrote a memo proposing an Internet-based hypertext system, which eventually led to the development of HTML

Is HTML case sensitive

HTML is not case sensitive. This means that you can use uppercase or lowercase letters interchangeably when writing HTML tags, attributes, and attribute values. However, it is considered a best practice to use lowercase letters for HTML elements and attributes to ensure consistency and readability in your code..
