JavaScript Review

PARAMETER

In JavaScript, a parameter is a variable that is listed as part of a function's declaration. It is used to receive values when the function is called or invoked. The values passed as arguments to the function are assigned to the corresponding parameters, allowing the function to work with different values each time it is called.

Who invented JavaScript?

JavaScript was invented by Brendan Eich. He created the language in 1995 while working at Netscape Communications Corporation. Initially, JavaScript was developed as a scripting language for web browsers to add interactivity and dynamic functionality to web pages. Over time, JavaScript has evolved and become one of the most widely used programming languages, not only for web development but also for server-side development, mobile app development, and more.

Basic structure

Structure

Statements: JavaScript code is written as a series of statements. Each statement performs a specific action or task.


Variables: Variables are used to store and manipulate data in JavaScript. They are declared using the var, let, or const keyword, followed by the variable name.


Data Types: JavaScript supports various data types, including numbers, strings, booleans, objects, arrays, and more. These data types define the kind of values that can be stored and manipulated in JavaScript.


Functions: Functions are reusable blocks of code that perform a specific task. They are defined using the function keyword, followed by the function name and a set of parentheses. Functions can accept parameters (inputs) and return values (outputs).


Control Flow: JavaScript provides control flow statements such as if statements, for loops, while loops, and switch statements. These statements control the flow of execution based on certain conditions or criteria.


Objects: JavaScript is an object-oriented programming language, and objects play a crucial role in JavaScript. Objects are collections of key-value pairs and can contain properties and methods.


Events: JavaScript can interact with HTML elements and respond to user actions or events. Event handlers can be attached to HTML elements to execute JavaScript code when a specific event occurs, such as a button click or form submission.
