In Python, a block of code is defined using indentation. Indentation is a way of visually grouping statements together to form a block of code. It is used to indicate the scope and hierarchy of code in Python. In most programming languages, blocks of code are defined using curly braces {} or keywords like begin and end. Get the data type of a variable



The type() function in Python is a built-in function that allows you to determine the data type of an object. It returns the class or type of the specified object.

The type() function takes an object as its argument and returns a type object that represents the data type of the object. This type object can be used to compare or check against other types.

For example, if you have a variable x and you want to know its data type, you can use the type() function like this: x = 10 print(type(x))
